Features

  • Parses rough ASCII & Unicode вҖ” tolerates misaligned corners, overflowing labels, broken edges
  • Preserves your layout вҖ” fixes alignment without rearranging boxes
  • 4 box styles вҖ” single в”Ңв”Җв”җ, rounded в•ӯв”Җв•®, double в•”в•җв•—, bold в”Ҹв”Ғв”“
  • Markdown mode вҖ” fix diagrams inside code blocks: bx fix --markdown README.md
  • Pipe-friendly вҖ” stdin/stdout, composes with other tools
  • Single binary вҖ” Go, no runtime dependencies

Install

go install github.com/justEstif/bx@latest

Usage

# Fix a diagram from stdin
cat rough.txt | bx fix

# Fix with rounded corners
echo '...' | bx fix --style rounded

# Fix diagrams inside markdown
bx fix --markdown README.md

# Fix a file directly
bx fix diagram.txt

Why?

LLMs and humans constantly produce broken ASCII diagrams вҖ” misaligned corners, off-by-one edges, inconsistent characters. bx takes that mess and snaps everything into geometrically perfect Unicode box-drawing. Works in terminals and markdown code blocks.
